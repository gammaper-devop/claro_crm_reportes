# Claro Collapse Component

## Usage

````html
<claro-collapse
    [text] = "collapse title"
    [disable] = "true | false"
    [expanded] = "true | false"
    [hideToggle] = "true | false"
    [iconName] = "icon name"
    (closed) = "closeAction()"
    (opened) = "openAction()"
    (iconClicked) = "iconClickAction()" 
></claro-collapse>
````
