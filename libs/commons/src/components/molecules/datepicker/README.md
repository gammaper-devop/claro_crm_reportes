# Claro Datepicker

## Usage 

````html
<claro-datepicker
  [placeholder] ="Seleccione"
  [type] = "text"
  [disabled] = "true | false"
  [value] = "string"
  [size] = "'md'"
  [icon] = "string"
  [minDate] = "Date | null"
  [maxData] = "Date | null"
  (valueChanged) = "onChangeAction()"
  (blurred) = "onBlurAction()"
  (enter) = "onEnter()"
  classes = "string[]"
></claro-datepicker>
````
