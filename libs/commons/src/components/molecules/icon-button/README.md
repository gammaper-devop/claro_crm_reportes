# Icon Button Component


## Usage 

`````html 
<claro-icon-button
  [name] = "button name"
  [type] = "submit | button"
  [color] = "primary | white"
  [styl] = "icon | flat | fab | basic"
  [size] = "sm | md | lg "
  [disabled] = "true | false"
  [text] ="button text"
  (clicked) = "onClickAction()"
  classes ="'{}'" 
>
</claro-icon-button>
````
