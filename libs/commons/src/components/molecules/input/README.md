# Claro Input Component

# Usage

````html
  <claro-input
   [placeholder] = "placeholder text"
   [type] ="text"
   [disabled] = "true | false"
   [value] = "value"
   [maxLength] = "'number'"
   [size] = "'md'"
   [icon] = "icon name | null"
   [paste] = "true | false"
   [inputValidation] = "number | alpha | alphanumeric | text"
   (valueChanged) = "onValueChange()"
   (blurred) = "onBlurAction()"
   (enter) = "onEnterAction()"
  ></claro-input>
````
